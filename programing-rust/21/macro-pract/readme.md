# Rust マクロ定義サンプル

`macro_rules!` を使った宣言マクロの基本パターンをまとめたサンプルです。

## 実行方法

```bash
cargo run
```

---

## サンプル一覧

### 1. `say_hello!` — 基本的な複数アーム

引数なし・あり の2つのアームを持つシンプルなマクロです。

```rust
say_hello!();           // => Hello, world!
say_hello!("Rust");     // => Hello, Rust!
```

**ポイント:** アームは `;` で区切り、パターンに合致した最初のアームが展開されます。

---

### 2. `my_vec!` — 繰り返し (`$(...)*`)

`vec!` マクロの簡易実装です。`$(...)*` で任意個数の引数を受け取ります。

```rust
let nums  = my_vec![1, 2, 3, 4, 5];
let words = my_vec!["foo", "bar", "baz",]; // 末尾カンマも OK
```

**ポイント:** `$(,)?` を末尾に付けると末尾カンマを許容できます。

---

### 3. `check_eq!` — アームのオーバーロード

2アームのマクロで、オプションのメッセージ引数を受け取ります。

```rust
check_eq!(1 + 1, 2);
check_eq!("hello".len(), 5, "文字列長が一致しない");
```

**ポイント:** `$msg:literal` はリテラル値（文字列・数値）のみ受け付けるメタ変数です。

---

### 4. `make_point!` — 識別子を受け取り構造体を自動生成

`$name:ident` で構造体名、`$field:ident: $type:ty` でフィールド定義を受け取り、構造体と `Display` 実装を生成します。

```rust
make_point!(Point2D, x: f64, y: f64);
make_point!(Point3D, x: f64, y: f64, z: f64);
```

**ポイント:** `stringify!($name)` でマクロ引数をそのまま文字列に変換できます。

---

### 5. `impl_describe!` — 複数の型に同じ `impl` を展開

複数の型をカンマ区切りで渡すと、それぞれに同じトレイト実装を生成します。

```rust
impl_describe!(i32, f64, bool);

42_i32.describe()   // => "42 (type: i32)"
3.14_f64.describe() // => "3.14 (type: f64)"
```

**ポイント:** `$($t:ty),+` で1つ以上の型をまとめて受け取れます。

---

### 6. `log_debug!` — 条件付きデバッグ出力

`cfg!(debug_assertions)` を使い、デバッグビルド時のみ出力するマクロです。

```rust
log_debug!("x = {}, y = {}", 10, 20);
// デバッグビルド => [DEBUG] x = 10, y = 20
// リリースビルド => 何も出力しない
```

**ポイント:** `$($arg:tt)*` は任意のトークン列を受け取る最も汎用的なメタ変数で、フォーマット文字列をそのまま内部マクロへ渡す際に使います。

---

### 7. `map!` — `HashMap` の簡潔な初期化

`key => value` というカスタム構文で `HashMap` を初期化するマクロです。

```rust
let scores = map! {
    "Alice" => 95,
    "Bob"   => 87,
    "Carol" => 92,
};
```

**ポイント:** `macro_rules!` はカスタム区切り文字（ここでは `=>`）を自由に定義できます。

---

## メタ変数の種類まとめ

| メタ変数 | 受け取るもの | 使用例 |
|----------|-------------|--------|
| `expr`   | 式          | `$x:expr` |
| `ident`  | 識別子（変数名・型名など） | `$name:ident` |
| `ty`     | 型          | `$t:ty` |
| `literal`| リテラル値  | `$msg:literal` |
| `tt`     | 任意のトークン | `$($arg:tt)*` |
| `pat`    | パターン    | `$p:pat` |
| `stmt`   | 文          | `$s:stmt` |
| `block`  | ブロック    | `$b:block` |

## 繰り返し修飾子まとめ

| 修飾子 | 意味 |
|--------|------|
| `*`    | 0回以上 |
| `+`    | 1回以上 |
| `?`    | 0回または1回 |
